# 🏥 Hospital Patient Care & Billing Analytics Dashboard

## 📌 Project Overview

This end-to-end Data Analytics project analyzes hospital patient records to uncover insights related to patient demographics, hospital performance, billing trends, insurance coverage, and healthcare operations.

The project follows a complete analytics workflow:

**Data Cleaning (Python) → EDA (MySQL) → Visualization (Matplotlib & Seaborn) → Interactive Dashboards (Tableau & Power BI)**
